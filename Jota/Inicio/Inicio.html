¡Bienvenido a Jota!

Gestiona de forma fácil e intuitiva tus imágenes, volúmenes, redes y contenedores docker.

En Jota queremos ofrecerte una forma gratuita y sencilla de sumergirte en el mundo de docker.

Jota es una una interfaz gráfica que gestiona imágenes, volúmenes, redes y contenedores docker de forma fácil e intuitiva.

Jota se apoya en las comunicaciones con la api de docker. Ésto se lleva a cabo mediante peticiones http que manda Jota a la api, a lo que ésta contesta con los datos en formato JSON y los sirve de forma que jota puede leer y tratar los datos que luego se reflejarán en la interfaz gráfica de una forma sencilla y atractiva para el usuario.
